# 30-Day Vibe Coding Dashboard

Minimal learning dashboard to track progress through the 30-day vibe coding roadmap. Mobile-first, clean UI, Next.js + SQLite + Tailwind.

## Stack

- Next.js (App Router)
- SQLite (file-based)
- Tailwind CSS
- Deploy: Vercel

No ORM, no auth, no extra abstraction.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
