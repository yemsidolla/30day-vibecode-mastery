# OpenClaw — AI Command Center Dashboard

A real-time AI operations hub where you monitor AI agents and sub-agents, visualize an AI office in motion, track projects and reports, chat/command your AI team, monitor Docker and servers, track scheduled tasks, enforce governance, and prevent system chaos.

**Vision:** "Jarvis meets DevOps meets CTO dashboard" — AI-first, visual, fun, strategic.

## How to use this repo

1. **Start at** [week-1/day-01/](week-1/day-01/) and work through each day in order.
2. **Follow** the [MASTER-PROMPT.md](MASTER-PROMPT.md) for every build step: design first, small chunks, STOP after each phase. Use GO NEXT / REFINE / AUDIT / SIMPLIFY as needed.
3. **Design** in [docs/](docs/) (architecture, schema, layouts, protocols). **Build** in [app/](app/) (backend + frontend).
